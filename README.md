# 🌦 WeatherWise

A simple desktop weather application built using Python and Tkinter.

## Features

- Search weather by city
- Live temperature
- Humidity
- Wind speed
- UV Index
- Chance of rain
- Local time
- Weather recommendation

## Technologies

- Python
- Tkinter
- Requests
- WeatherAPI

## Installation

```bash
pip install requests
python main.py
```
